Algiz rune for protection

Rönd is a true shield that enables you to improve data security while providing full independence and 360-degree permission governance.

Rönd is an authorization mechanism that defines the concepts of Roles, Permissions, and User Groups as building blocks.

These building blocks allow you to create your access-security model based on the real function that people have in your organization, rather than the specific actions they need to perform.
